# CompuWork - Sistema de Gestión de Recursos Humanos

## Cómo compilar y ejecutar la prueba

1. **Abre tu terminal** y navega hasta la carpeta exacta donde guardaste todos los archivos `.java`.
2. **Compila** el código fuente ejecutando el siguiente comando (esto creará los archivos `.class` listos para ejecutarse):
   ```bash
   javac *.java
   ```
3. **Inicia el programa** ejecutando **Main**
   ```bash
   java Main
   ```

## Resultado Esperado 

- Mensajes de éxito confirmando la asignación de empleados al departamento.

- El manejo de errores si se intenta registrar un puntaje inválido o duplicar un empleado.

- Reportes de desempeño individuales con el cálculo exacto de la compensación para cada tipo de empleado.

- El reporte consolidado del departamento con su promedio de rendimiento.
